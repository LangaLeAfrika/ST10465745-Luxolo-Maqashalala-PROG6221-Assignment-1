# CyberSecurityBot 🤖

## 📌 Overview

CyberSecurityBot is a C# console application that simulates a basic chatbot focused on cybersecurity awareness and user interaction.

---

## ⚙️ Features (Version 1)

* ASCII art welcome screen
* Audio greeting using `.wav` file
* User name input and validation
* Interactive chatbot conversation
* Typing effect for realistic responses

---

## 🧠 Technologies Used

* C#
* .NET Console Application
* System.Media (audio playback)
* System.Threading (typing effect)

---

## ▶️ How to Run

1. Open the project in Visual Studio
2. Ensure `greeting.wav` is in the output directory
3. Run the application

---

## 📂 Project Structure

* `Program.cs` – Entry point
* `UIHelper.cs` – UI display and effects
* `AudioPlayer.cs` – Handles audio playback
* `ChatBot.cs` – Chat logic
